# Getting Started

## Requirements

* Node.js (LTS recommended)
* Git

## Install

1. Clone repo
2. Install deps
3. Run dev server

> Replace commands below with the actual package manager once chosen.

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Dev Notes

* Keep assets small for fast iteration
* Prefer glTF/GLB for models
