# Town Block Art Spec

## Scale

* 1 unit = 1 meter
* Player height ~1.7–1.8 units

## Layout Goals

* One loop path
* One focal square
* One interior
* 2–3 landmarks

## Collider Authoring

* Use simple boxes for buildings
* Name colliders with prefix: `COL_`

## Export

* Export as `.glb`
* Apply transforms
* Keep textures reasonable (2K max for MVP)
