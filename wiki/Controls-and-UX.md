# Controls & UX

## Desktop (MVP)

* **WASD**: move
* **Mouse**: camera orbit
* **Shift**: sprint (optional)
* **Space**: jump (optional)
* **E**: interact
* **1/2**: day/night toggle

## UX Principles

* Prompts are minimal, diegetic when possible
* Keep HUD small: crosshair (optional), interact prompt, phase indicator
* Debug UI is separate and easy to disable
