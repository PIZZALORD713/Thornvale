# Game Loop

## Day (Cozy Mode)

* Explore town
* Meet villagers
* Run errands (simple quests)
* Light crafting/foraging (optional for early builds)
* Build “Neighborliness” (trust)

## Night (Uncanny Mode)

* Investigate anomalies
* Learn rules you weren’t told
* Experience “polite interventions” when you break patterns
* Progress the mystery

## Progression

* Cozy upgrades (tools, access)
* Uncanny resistance (perception, courage, clarity)
